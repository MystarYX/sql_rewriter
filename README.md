# sql_rewriter

基础版 SQL 字段分析工具（V1）。

## 功能
- 旧代码分析 / 标准代码分析双入口
- SQL 解析（`AS` 与无 `AS` 别名）
- 提取来源表（`FROM`）
- 结果表格展示：来源表、原始字段、映射字段、注释
- 空输入与解析失败提示
- 复制结果（TSV）

## 运行
1. 直接用浏览器打开 `index.html`
2. 粘贴 SQL，点击“解析”

## 测试
```bash
node --test ./tests/parser.test.js
```

## 说明
- 当前版本优先支持基础 `SELECT ... FROM ...` 映射解析。
- 子查询、复杂方言函数、深层嵌套为后续迭代范围。
