# sql_rewriter

SQL 字段血缘解析 + 标准字段重构工具（V2）。

## 主要能力
- SQL 结构解析（SELECT / FROM / JOIN / WITH CTE / 子查询 / UNION 基础支持）
- 字段血缘关系输出（源表/源字段 -> 目标表/目标字段）
- 标准字段字典替换并生成新 SQL
- 字段重构报告输出（旧字段/新字段/位置）

## 页面功能
- 双入口：旧代码分析 / 标准代码分析
- SQL 输入 + 标准字段字典输入
- 一键解析并输出：
  - 字段映射清单
  - 血缘关系表
  - 标准化 SQL
  - 重构报告

## 运行
1. 浏览器打开 `index.html`
2. 输入 SQL 与字典（每行 `旧字段,新字段`）
3. 点击“解析并重构”
